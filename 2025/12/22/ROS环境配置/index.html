ROS环境配置

一、环境配置

1、ubuntu安装

​ Ubutnu 与 Windows 一样都是电脑操作系统，而目前 ROS 主流都是在 Ubutnu系统上运行的。
本资料绝大部分内容也都是基于 Ubutnu 的。所以在进行 ROS 学习之前，必须要先学会 Ubuntu 系统相关的知识。使用 Ubuntu 系统，用户可以给自己的电脑安装一个 Ubutnu 系统，这个操作难度比较高。所以我们在此提供一个更简单的方法，就是使用 VMware 虚拟机，只需要在 windows 系统上安装 VMware 软件就可以在windows 系统上使用Ubuntu 系统。
​ ROS发展至今，有Melodic，Kinetic，Indigo等等，不同版本对应不同版本的ubuntu。本次教程采用的是20.04版本的ubuntu，对应的是ROS版本为noetic。由于国外源ubuntu下载速度慢，此处我们采用国内镜像源进行下载：清华镜像源

3、ROS应用商店

当我们安装好ROS后，首先我们要学会下载并阅读别人的代码。常用的两个应用商店一个是ROS的官方应用商店ROS index，另一个就是我们熟知的github了。接下来我将分别以ROS index与github分别教学如何安装程序。

这里我们先打开浏览器输入网址 https://index.ros.org/ （maybe需要加速器）这里就是ROS的官方应用商店。
